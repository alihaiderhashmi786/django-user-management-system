# Django User Management System

A simple Django project using MySQL (XAMPP) to manage users.

## Features

- Create User
- Store Data in MySQL
- View All Users
- Filter Users by Age
- Simple and Clean UI

## Technologies Used

- Python
- Django
- MySQL (XAMPP)
- HTML
- CSS

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a MySQL database.

4. Update database settings in `settings.py`.

5. Run migrations

```bash
python manage.py migrate
```

6. Start the server

```bash
python manage.py runserver
```
